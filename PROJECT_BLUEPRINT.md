# University Training Management System

## Project Vision

نظام أكاديمي وتدريبي احترافي متعدد المؤسسات التعليمية في العراق.

يدعم:

- الجامعات الحكومية
- الجامعات الأهلية
- الكليات
- المعاهد
- مراكز التدريب

يعمل باللغتين:

- العربية
- الإنجليزية

ويعتمد تصميماً حديثاً وقابلاً للتوسع.
